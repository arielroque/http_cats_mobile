## :triangular_flag_on_post: Running App

```bash
#Clone the project
git clone https://github.com/arielroque/http_cats_mobile.git

#Enter in the folder 
cd http_cats_mobile

#Get the packages
flutter packages get

#Running the app in your device ou emulator
flutter run
```
